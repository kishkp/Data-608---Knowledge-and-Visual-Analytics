Chart Details: In this chart, we can analyze the trend of complaints, disputes and timely resolutions across different time period. Setting the X-Axis to "Time", allows us to see a time-lapse display of the trends for the respective categories over time.

Dimensions: The dimension for the chart can be choosen with the below radio-buttons. For example, if we select 'Product' as a dimension, then in the chart we have an option to select the respective product that we can show the trend for.
